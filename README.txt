Written with Python 3.8.10

This simple script converts a video file to a gif animation. It uses ffmpeg
library.

Usage:
    video2gif.py <video filename> [fps]

Command-line parameters:
    <video filename> - input video file to be converted to gif;

    [fps] - frames per second for gif animation (optional).

Feel free to use and modify it as needed.

For dependencies refer to requirements.txt.
